Financial Formulas & Calculations

Return on Equity (ROE): $$ROE = \frac{\text{Net Income}}{\text{Shareholders' Equity}} \times 100$$

Capital Adequacy Ratio: $$CAR = \frac{\text{Tier 1 Capital + Tier 2 Capital}}{\text{Risk-Weighted Assets}} \times 100$$

Net Present Value (NPV): $$NPV = \sum_{t=0}^{n} \frac{R_t}{(1+i)^t}$$ where $R_t$ = net cash flow, $i$ = discount rate, $t$ = time period

These are the core financial formulas we use in our analysis. Our ROE this quarter is 18.7%, well above industry average of 12%. CAR stands at 14.2%, exceeding regulatory requirements of 10.5%. NPV calculations drive our investment decisions.

Data Analytics Query


-- Customer Lifetime Value Analysis
WITH customer_metrics AS (
    SELECT 
        customer_id,
        SUM(transaction_amount) as total_revenue,
        COUNT(DISTINCT transaction_date) as engagement_days,
        DATEDIFF(MAX(transaction_date), MIN(transaction_date)) as customer_age_days
    FROM transactions
    WHERE transaction_date >= '2025-01-01'
    GROUP BY customer_id
)
SELECT 
    cm.customer_id,
    cm.total_revenue,
    cm.total_revenue / NULLIF(cm.customer_age_days, 0) * 365 as annual_clv,
    c.segment,
    c.acquisition_channel
FROM customer_metrics cm
JOIN customers c ON cm.customer_id = c.id
WHERE cm.total_revenue > 1000
ORDER BY annual_clv DESC
LIMIT 100;
This query calculates Customer Lifetime Value for our top customers. We use this for segmentation and targeted marketing campaigns. The analysis helps identify high-value customers for retention programs.

Value at Risk (VaR) Calculation

Historical Simulation Method:

$$\text{VaR}_\alpha = -\inf\{x \in \mathbb{R} : P(X \leq x) > \alpha\}$$

For a 95% confidence level portfolio: $$\text{VaR}_{0.05} = \mu - 1.645\sigma$$

Expected Shortfall (Conditional VaR): $$\text{ES}_\alpha = \mathbb{E}[X | X \leq \text{VaR}_\alpha]$$

VaR measures potential loss in portfolio value. We use 95% confidence level for standard risk assessment. Expected Shortfall provides additional tail risk information. These metrics guide our risk management strategies.
